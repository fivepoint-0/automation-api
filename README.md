# automation-api

To install dependencies:

```bash
make install
```

To run:

```bash
make run
```

To run tests:
```bash
make test
```

To build:
```bash
make build
```

To tag a build with a version number (like `1.1.0`):
```bash
IMAGE_TAG=1.1.0 make tag
```

To start container:

```bash
make container
```
